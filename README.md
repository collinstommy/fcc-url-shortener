# fcc-url-shortener

Pass in a full length url. A shortened url will be returned

# Usage
* /api/new/http://www.freecodecamp.com

#Output
```javascript
{"original_url":"https://www.google.com","short_url":"localhost:8080/1"}
```

# Usage
* /1

#Redirects to 
```javascript
"https://www.google.com"
```
